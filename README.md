# 🌊 河道水质智能评估平台

> **浙江省大学生物理实验与科技创新竞赛 · 职教赛道**
> 
> **主题：物理学与AI赋能浙江省科技发展**

## 📋 项目简介

本项目开发了一套基于**多模态AI与光学特征分析**的河道水质智能评估平台。系统通过普通摄像头采集水体图像，利用计算机视觉技术提取水体颜色、浊度、透明度、悬浮物等**7类26维光学物理特征**，结合**RandomForest机器学习模型**自动判断水质等级，并生成智能检测报告。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| 📷 图像采集 | 支持图片上传和摄像头拍摄 |
| 🔬 光学特征提取 | 7类26维光学物理特征自动提取 |
| 🤖 AI分类 | RandomForest模型自动判断水质等级 |
| 📊 结果展示 | 概率分布、雷达图、特征贡献分析 |
| 📑 AI报告 | 政府环保监测报告级别（可选大模型增强） |
| 📥 报告导出 | JSON/CSV/完整报告下载 |

## 🚀 快速开始

### 环境要求

- Python 3.8+
- 4GB RAM（推荐）
- 摄像头（可选，用于实时拍摄）

### 安装步骤

#### 第一步：克隆项目

```bash
git clone <项目地址>
cd river-water-quality-assessment
```

#### 第二步：安装依赖

```bash
pip install -r requirements.txt
```

#### 第三步：训练AI模型

```bash
python train_model.py
```

训练过程将：
1. 自动生成2000条合成训练数据（4类水质各500条）
2. 训练RandomForest分类模型
3. 保存模型文件到 `models/` 目录
4. 输出模型评估报告

#### 第四步：启动应用

```bash
streamlit run app.py
```

应用将在浏览器中打开：`http://localhost:8501`

## 📖 使用指南

### 基础使用

1. 在左侧边栏设置监测位置和水体类型
2. 上传水体图片或使用摄像头拍摄
3. 点击"开始水质检测分析"按钮
4. 查看检测结果和分析报告

### 大模型增强（可选）

系统支持集成大语言模型生成政府环保监测报告级别的分析文档。

#### 配置方式一：本地LM Studio

1. 启动LM Studio并加载模型（推荐 Qwen2.5 7B）
2. 在本地服务器中启动API服务
3. 在应用侧边栏中启用"AI报告增强"
4. 填写API地址：`http://localhost:1234/v1`

#### 配置方式二：OpenAI API

1. 在侧边栏填写API地址：`https://api.openai.com/v1`
2. 填写API密钥
3. 填写模型名称：`gpt-3.5-turbo` 或 `gpt-4`

#### 配置方式三：环境变量

```bash
export LLM_API_KEY="your-api-key"
export LLM_API_BASE="http://localhost:1234/v1"
export LLM_MODEL="qwen2.5-7b-instruct"
```

### 报告导出

检测完成后可导出：
- **JSON格式**：完整的检测数据
- **CSV格式**：26维光学特征数据
- **Markdown格式**：含AI分析的完整报告

## 🔬 技术原理

### 光学特征提取（7类26维）

| 特征类别 | 维度 | 物理意义 |
|---------|------|---------|
| RGB颜色特征 | 6 | 反映水色（蓝→绿→黄→褐） |
| HSV颜色特征 | 6 | 反映色相/饱和度/明度 |
| 图像亮度 | 1 | 反映透光性 |
| 图像对比度 | 1 | 反映水体均匀性 |
| 浑浊度指标 | 4 | 基于光学原理的浑浊度评估 |
| 悬浮颗粒特征 | 4 | 基于边缘检测的颗粒分析 |
| 纹理特征 | 4 | 基于GLCM的纹理分析 |

### AI分类模型

- **模型**：RandomForest（随机森林）
- **参数**：200棵决策树，最大深度15
- **特征**：26维光学特征
- **输出**：优/良/中/差 + 置信度
